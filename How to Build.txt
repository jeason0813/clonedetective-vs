How to Build Clone Detective
============================

Prerequisites
-------------

1. Microsoft Visual Studio 2008 (Standard, Professional or any edition of the Team Suite).

2. Microsoft Visual Studio 2008 SDK
   http://www.microsoft.com/downloads/details.aspx?familyid=30402623-93ca-479a-867c-04dc45164f5b

3. Windows Installer XML (WiX) toolset Version 3
   http://wix.sourceforge.net/

4. ConQAT 1.3, download and copy to Lib\ConQAT
   http://conqat.cs.tum.edu/index.php/Download

5. Sandcastle
   http://www.codeplex.com/Sandcastle

6. Sandcastle Help File Builder (SHFB)
   http://www.codeplex.com/SHFB

7. Sandcastle Extensions
   http://www.codeplex.com/SandExtensions

Building
--------

1. Run Build.bat (make sure the path to MSBuild is correct)

2. The folder Output will now contain the following subfolders:
   - Bin. Contains the binaries (compiled in "Release" configuration).
   - Help CHM. Contains the standalone help file.
   - Help VS. COntains the help files needed for Visual Studio help integration.
   - Setup. Contains the setup.   
   - Source. Contains the complete source code (without TFS bindings).
